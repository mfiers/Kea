Welcome to the Mad/Kea/Moa documentation
====================================

Mad/Kea/Moa is a system to improve your life as a command line bioinformatician. In consists of three components:

  * Mad - store file metadata
  * Kea - supercharge command line tools, store provenance
  * Moa - (very) simple workflows


The project is under development / documentation will grow in the future.
